# Perky News Contracts

Smart contracts for Perky News premium subscriptions and on-chain features.

## Planned Features

- **Subscription NFT**: Premium subscribers receive an NFT as proof of subscription
- **x402 Integration**: On-chain payment verification
- **Access Control**: Token-gated content access

## Tech Stack (Planned)

- **Framework**: Foundry
- **Network**: Base / Base Sepolia
- **Standards**: ERC-721 (Subscription NFT)

## Setup

```bash
# Install Foundry
curl -L https://foundry.paradigm.xyz | bash
foundryup

# Build
forge build

# Test
forge test
```
